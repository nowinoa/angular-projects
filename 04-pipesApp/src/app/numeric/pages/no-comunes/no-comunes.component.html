value_expression | i18nPlural : pluralMap [ : locale ]

Slice Pipe: sirve para cortar la información que se desea mostrar. El primer valor (start) sirve para indicar en que posición comienza y el segundo valor (end) para terminar.

value_expression | slice : start [ : end ]
{{ clientes }}
Ej. slice:0:2
{{clientes | slice:0:2}}
Ej. slice:2:6
{{clientes | slice:2:6}}

Si pulsas el botón de borrar cliente, debe eliminarse un nombre.

Json Pipe: sirve para mostar código.

value_expression | json

{{heroes | json}}

KeyValue Pipe: sirve para mostrar valores de un arreglo.

input_expression | keyvalue [ : compareFn ]
  • {{item.key | titlecase}} : {{item.value}}

Async Pipe: nos permite visualizar en pantalla el contenido de una observable o una promesa.

Utilizar async pipe para la ausencia de un valor mediante ngIf

Por ejemplo: para mostrar un mensaje mientras obtenemos la información necesaria.

Para la negación debemos utilizar !(#)
